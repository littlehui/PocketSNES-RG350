金手指使用:
将 "游戏名.cht"金手指文件放到  .pocketsnes/cheats 目录下即可。
金手指文件格式：

--------------------------------------------------
[1P命/B]
锁=7e1F8A,09

[无敌]
锁=7e1F89,01

[1#武器选择]
初始=7e1F84,00
散弹=7e1F84,01
爆弹=7e1F84,02
追弹=7e1F84,03
火焰=7e1F84,04
激光=7e1F84,05



[GameInfo]
Name=魂斗罗三代之异型入侵
System=SFC
Text=SNEShout模拟器测试有效
